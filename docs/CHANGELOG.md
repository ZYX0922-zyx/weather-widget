# Changelog

## [1.0.0] - 2026-05-31

### Added
- Transparent desktop weather widget for Windows (Electron)
- Desktop wallpaper layer mount via WorkerW
- Normal / vertical strip / horizontal compact layout modes
- City search, geolocation, IP fallback, local SVG weather icons
- DPI-aware drag and native Win32 resize
- One-click install scripts and bilingual UI (en / zh-CN)
- Troubleshooting documentation from full development history

### Fixed
- Mouse capture stickiness after resize/drag
- Startup crashes and invisible window issues
- Strip mode reverting to normal window after shrink
- Drag flicker and command window not exiting on close
